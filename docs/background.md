
# 背景
## 摘要:
描述一个因癌症化疗脱发的美少女美杜莎拯救世界的故事.

## 故事: 
美杜莎是一个可爱的小女孩,她曾经有一头茂密可爱的头发,快乐地和家人与朋友度过每一天.但突如其然的癌症打破了她的美好的生活,化疗使她失去了美丽的头发,使她被困在医院,但家人和朋友没有放弃美杜莎.他们的关切使美杜莎乐观的面对病魔.

但祸不单行,来自未知魔法世界的污染,世界产生了很多怪物,他们攻击人类的城市,使得人们生病与死亡.

还是攻击到了美杜莎所在的医院,身残志坚的美杜莎为保护朋友,勇敢的面对怪物,唤醒并打动了在医院的健体之蛇,他消灭了攻击医院的怪物,并化成美杜莎的头发,美杜莎从此获得驱使魔法的能力,变成了传说中的魔法美少女.

看着受伤的家人与朋友们,健体之蛇说,想要拯救家人,朋友,必须找到获得其他能力的魔力蛇,强大美少女的魔力,对抗与日俱增的魔力怪物.方能让家人与朋友恢复原本的生活,于是美杜莎撑起勉强的身体,开始探索之路



## 蛇的种类:
紧扣主题 Python
健体之蛇: 增加体质点 使得主角可以下床,缓慢的运动
寒冰之蛇: 使周围环境的水结冰,攻击怪物,阻碍怪物移动
聚毒之蛇: 使全部蛇的毒聚集起来,是大招毒灭怪物,净化世界的关键
弹跳之蛇: 使主角获得跳跃的能力
迅捷之蛇: 使主角获得奔跑的能力
